# WIPULSCAN PRO v6.0 - GitHub Pages Deployment

## Schritt-für-Schritt Anleitung

### 1. GitHub Repository erstellen
1. Gehe zu https://github.com/new
2. Repository-Name: `wipulscan-pro`
3. Wähle "Public"
4. Klicke "Create repository"

### 2. Dateien hochladen
1. In deinem neuen Repository klicke auf "Upload files"
2. Ziehe alle Dateien aus diesem Ordner (`github-pages/`) in das Upload-Feld
3. Warte bis alles hochgeladen ist
4. Klicke "Commit changes"

### 3. GitHub Pages aktivieren
1. Gehe zu "Settings" (oben rechts)
2. Im linken Menü klicke auf "Pages"
3. Unter "Source" wähle "Deploy from a branch"
4. Wähle "main" und "/(root)"
5. Klicke "Save"

### 4. Fertig! 🎉
Nach 1-2 Minuten ist deine App live unter:
```
https://DEIN-USERNAME.github.io/wipulscan-pro
```

## Deine öffentliche URL
Ersetze `DEIN-USERNAME` mit deinem GitHub-Namen:
- **https://DEIN-USERNAME.github.io/wipulscan-pro**

## Für Handy (PWA)
1. Chrome auf Handy öffnen
2. URL eingeben: `https://DEIN-USERNAME.github.io/wipulscan-pro`
3. Chrome-Menü → "Zum Startbildschirm hinzufügen"
4. Fertig! 📱
