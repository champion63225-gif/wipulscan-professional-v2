SCHNELLSTART:
============

1. DOPPELKLICK auf "STARTEN.bat"
   → Ein schwarzes Fenster öffnet sich
   → Der Server startet automatisch
   → Der Browser öffnet sich mit der App

2. FÜR HANDY/ANDEREN PC:
   → Im schwarzen Fenster steht deine IP-Adresse
   → Auf Handy: Chrome öffnen
   → Adresse eingeben: http://[DEINE-IP]:3000
   → Bsp: http://192.168.1.100:3000

3. ALS APP INSTALLIEREN (Handy):
   → Chrome-Menü (3 Punkte)
   → "Zum Startbildschirm hinzufügen"
   → Fertig!

DATEIEN:
========
- index.html      = Die Hauptseite
- STARTEN.bat     = Einfacher Start-Knopf
- jingle-intro.mp3 = Sound
- assets/         = Alle Bilder und Scripts

HILFE:
======
Wenn nichts geht:
1. Node.js muss installiert sein (nodejs.org)
2. Internet für erste Installation nötig
3. Firewall könnte blocken - dann Port 3000 freigeben
